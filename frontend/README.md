# LinX (灵枢) - Frontend

React + TypeScript + Vite frontend for LinX (灵枢) intelligent collaboration platform.

## Quick Start

```bash
cd frontend
npm install
npm run dev
```

Visit http://localhost:3000

## Build

```bash
npm run build
```

## Tech Stack

- React 18 + TypeScript
- Vite
- TailwindCSS
- React Router
- Zustand
- i18next
- Axios
