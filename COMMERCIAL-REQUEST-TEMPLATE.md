# LinX 商业使用报备模板

本模板用于提交 LinX 商业使用报备申请。

申请方可直接复制以下内容，通过项目维护者公开的商务渠道或私有联系方式提交。

## 报备模板

```text
主题：LinX 商业使用报备申请 - <公司名/团队名> - <项目名>

我们计划在以下项目中使用 LinX，现根据 LICENSE 与商业使用说明进行报备：

1. 使用主体
- 公司 / 团队名称：
- 联系人姓名：
- 联系邮箱：
- 联系电话 / IM：

2. 项目信息
- 项目名称：
- 项目类型：<客户项目 / 内部生产系统 / SaaS / 商业产品 / 其他>
- 项目简介：
- 预计上线时间：
- 使用范围：

3. 使用方式
- 是否会修改源码：
- 是否会对外提供服务：
- 是否面向客户交付：
- 是否用于内部生产环境：
- 预计部署数量 / 环境数量：

4. 技术信息
- 计划使用的模块：
- 部署环境：
- 主要模型提供方：
- 是否使用外部运行节点：

5. 商业授权确认
- 我们已知悉 LinX 当前商业使用规则为：每个项目在使用前需先报备，并缴纳 150 美元声明费用。
- 我们希望获得该项目的商业使用许可。

6. 其他补充说明
-
```

提交信息应至少包含使用主体、项目范围、使用方式与部署形态等基础信息，以便维护者完成授权确认。
